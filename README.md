# NGI-GM-Dev-App
Notre projet consiste à faire deux taches principales :

1ere tache : un chronomètre qui permet de calculer le temps effectué durant un trajet ou un voyage, il se déclenche automatiquement une fois le conducteur commence son trajet (Pour pouvoir tester : donnez votre localisation en modifiant dans {position panel} les parametres latitude et longitude et fixez la vitesse moyenne à une valeur{average_speed} ).

2eme tache : notre application permet  d'afficher les pharmacies proches de votre destination.
